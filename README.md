# CS:APP Labs 通关记录

本项目包含《深入理解计算机系统》(CS:APP) 核心实验的代码实现，目前已完成前四个实验，均满分通过测试。后续实验还在继续做，会持续更新...

## 实验进度与总结

### 1. Data Lab
* 在严格的操作符数量和禁用规则限制下，仅使用基础位运算实现了整数和浮点数的底层操作。
* 深入理解了 C 语言隐式的优先级规则，剥离了非必要的括号，使代码结构更贴近底层逻辑与极简风格。

### 2. Bomb Lab
* 纯熟使用 GDB 进行二进制程序的逆向工程调试。
* 成功反推了多重逻辑分支，特别是从控制流中逆向还原了 Phase 6 的链表重排序逻辑，并解开了隐藏的二叉树递归关卡 (Secret Phase)。

### 3. Attack Lab
* 剖析了程序的运行时栈结构，通过栈溢出 (Buffer Overflow) 实现了代码流劫持。
* 在开启 ASLR 和栈区不可执行的防御机制下，构造了 Return-Oriented Programming (ROP) 链，计算栈偏移动态传入字符参数，成功绕过了保护机制。

### 4. Cache Lab
* **Part A (LRU Cache Simulator)**：使用 C 语言实现了一个支持自定义参数的缓存模拟器，并针对 LRU 替换策略的更新操作进行了复杂度优化。
* **Part B (Matrix Transpose)**：深入优化了数据空间局部性。对 64x64 矩阵创造性地采用了 4x4 子块切分，利用目标矩阵临时缓存数据并手动展开循环，将 Cache Conflict Miss 压到了理论最低区间。
