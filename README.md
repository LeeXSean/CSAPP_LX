# CS:APP Labs 通关记录

本项目包含《深入理解计算机系统》(CS:APP) 核心实验的代码实现，目前已完成前四个实验，均满分通过测试。后续实验还在继续做，会持续更新...

## 实验进度与总结

### 1. Data Lab
* 在运算符数量和禁用规则限制下，仅使用基础位运算实现整数和浮点数的底层操作。
* 熟悉 C 语言位运算符号的优先级规则，去除非必要的括号，使代码结构更简单。

### 2. Bomb Lab
* 学习使用 GDB 进行二进制程序的逆向工程调试。
* 成功反推多重逻辑分支，并解开隐藏的二叉树递归关卡 (Secret Phase)。

### 3. Attack Lab
* 剖析程序的运行时栈结构，通过栈溢出 (Buffer Overflow) 实现代码流劫持。
* 在开启 ASLR 和栈区不可执行的防御机制下，构造 Return-Oriented Programming (ROP) 链，计算栈偏移动态传入字符参数，成功绕过保护机制。

### 4. Cache Lab
* **Part A (LRU Cache Simulator)**：使用 C 语言实现一个支持自定义参数的缓存模拟器，并针对 LRU 替换策略的更新操作进行复杂度优化。
* **Part B (Matrix Transpose)**：优化数据的空间局部性。对于 64x64 矩阵，将每个 8x8 块内部切分为四个 4x4 子块，将目标矩阵 B 尚未写入的区域作为临时缓存。
